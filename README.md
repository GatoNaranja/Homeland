### 什么是Homeland项目
Homeland是斯皮克斯社区论坛的名称，Homeland项目就是斯皮克斯社区论坛手机客户端。Homeland是斯皮克斯金刚鹦鹉吧吧友的家，是安全的避风港，Homeland项目也将面向所有论坛坛友开源。
